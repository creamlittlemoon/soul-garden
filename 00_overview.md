# 00 — Product Overview

## 1. 产品一句话
**Soul Garden**：一个偏疗愈/哲学/艺术气质的自我探索 App。用户通过“喜欢的文字”来培育内在花园，并生成“气味人格画像”（内在气质的外显）。

## 2. 核心卖点（3 件事）
1) **可见的成长**：树苗具象成长（stage）+ 花园环境逐步丰盈  
2) **微仪式感**：每日一句 + 呼吸感动效 + 轻音景  
3) **独特差异点**：Words → Temperament → **Scent Identity**（气味人格）

## 3. MVP 目标（必须跑通）
- 4 个核心页面：Opening / Garden Home / Explore / Profile
- 内容库：≥ 1000 条（以原创为主，公版为辅，少量短引用）
- 三个内在属性：Calm / Courage / Clarity
- 气味标签体系：20–40 个 scent tags
- 树苗成长：5 个 stage
- 订阅付费墙：轻量（不影响体验，但可付费解锁扩展）

## 4. 用户主循环（Core Loop）
Open → Daily Whisper → Garden → Explore quotes → Like/Save → Attribute+XP → Tree grows → Scent profile updates → (return tomorrow)

## 5. 关键假设（用于验证）
- 用户愿意每天打开 1–3 分钟做“轻仪式”
- 用户会对“翻面看标签 + 气味画像”产生好奇与持续性
- 订阅的付费点来自：**无限探索 + 花园主题 + 气味报告 + 月度回顾**

## 6. 地区策略（默认）
- **海外 iOS 优先**（订阅习惯 + 审核风险更低）
- 中国大陆：后续用内容号/小程序做轻量引流版本（非 MVP）
