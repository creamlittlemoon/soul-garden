# Soul Garden — OpenClaw Markdown Pack (v0.2)

本压缩包是给 **OpenClaw Agent** 用的多文件 Markdown 规格说明，目标是让 Agent **不需要额外追问**即可开始实现与迭代。

## 已锁定的产品选择
- App 名：**Soul Garden**
- 视觉风格：**幻想风 / 朦胧 / 空灵**（风格 C）
- Explore 交互：**点击式卡片**（不是左右滑）
- 内容来源：**原创 + 书摘**（以版权安全策略落地）
- 香味系统：**人格气味（Scent Identity）**
- 成长形态：**具象植物成长（种子→树苗→树→开花→森林）**
- App 气质：**疗愈 + 哲学 + 艺术**（ABC 结合）

## 文件导航
1. `00_overview.md`：定位、用户旅程、MVP 边界、关键假设  
2. `01_experience_style_guide.md`：**风格圣经**（音乐/背景/字体/动效/粒子/颜色/触感/节奏）  
3. `02_ia_screens.md`：信息架构、页面清单、每屏 UI 组件与状态  
4. `03_systems_growth_scent.md`：成长系统 + 属性系统 + 气味人格算法（MVP→迭代）  
5. `04_content_copyright.md`：内容生产、标签体系、版权与引用策略（可落地）  
6. `05_data_api_schema.md`：Supabase 表结构、关键接口、事件埋点  
7. `06_monetization_release.md`：订阅与付费墙、地区策略（海外优先）、发布清单  

> 备注：本包偏“规格与体验”，不展开逐行实现步骤；OpenClaw 可自行选择 RN/Expo + Supabase 作为默认技术栈。
