# 03 — Systems: Growth + Attributes + Scent Identity

## 1. Growth（XP & Stage）
### XP 规则（MVP）
- Like: +5 XP
- Save: +8 XP
- Daily visit: +3 XP

### Stage 阈值（可调）
- seed → sprout: 50 XP
- sprout → young tree: 200 XP
- young tree → bloom: 500 XP
- bloom → forest: 1000 XP

## 2. Attributes（内在属性）
### MVP 三属性
- Calm / Courage / Clarity

### 计分规则（MVP）
- 每条 quote 带 attribute_tags（1–2）
- Like：对应属性 +1
- Save：对应属性 +1（或额外 +1）

## 3. Scent Identity（气味人格）
### 3.1 Scent tags（建议 24 个）
woody, earthy, petrichor, amber, musk, clean, citrus, herbal,
floral, sweet, smoky, spicy, tea, rice, ocean, green,
powdery, leather, incense, vanilla, cacao, coffee, mint, fig

### 3.2 计分规则（MVP）
- 每条 quote 带 2–4 个 scent tags
- Like：每个 tag +1
- Save：每个 tag +2

### 3.3 画像生成（MVP）
- Top 3 scent tags → 生成 Name + 1 句描述（非诊断）

Name 示例：Forest Rain / Moonlit Linen / Quiet Amber
